# React 18.2 Typescript, Webpack 5 Bundle

## Kits standard initial project

**React 18.2**
**Webpack 5**
**Typescript 4.6**
**ESLint 7**
**Tailwindcss 3**
**Postcss 8**
**Antd 5**
**MQTT 4**
**redux **
**@reduxjs/toolkit **

### > Require project:

node version 18.x
yarn or npm cli

### > Install:

```
npm install
```

or with Yarn

```
yarn install
```

### > Development (Local):

```
npm run start
```

or with Yarn

```
yarn start
```

### > Build (Production):

```
npm run build

```

or with Yarn

```
yarn run build

```
